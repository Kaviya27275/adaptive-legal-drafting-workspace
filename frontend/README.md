# Lexora — Frontend

This is a minimal Vite + React frontend scaffold for the Lexora project. It provides basic routing, a draft editor that saves to localStorage, and a simple login stub.

Quick start:

1. From the `frontend` folder install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

3. Open http://localhost:5173

Notes:
- The API base URL in `src/api/axiosConfig.js` is empty — set it to your backend URL when available.
- This scaffold focuses on making the app run locally; further feature work (AI suggestions, clause manager, etc.) can be added next.
